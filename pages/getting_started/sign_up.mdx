## Introduction

In order to lessen the effort required by user which may in turn encourage participation, user accounts are automatically created by the stakeholders and companies utilising the service.
After the initial onboarding using a company generated password, users will change passwords for security sake.

## Features

- User registration with required information by organization.
- Secure password hashing and validation at the first connection.
- Email verification to ensure account authenticity.
- Error handling for registration failures.
- Integration with the user management system of the "Here" platform.
- Responsive design for various devices.
